# WhatsApp Bot User Guide: Stunting Education, Self-Efficacy & Parenting Stress

This guide covers every feature of the bot, step by step.

The bot communicates in Indonesian.  All menus, options, and responses appear in Indonesian.

---

## Getting Started

1. Send any message to the bot number.
2. The bot replies with a welcome message and two poll options:
   - Ya, saya setuju (Yes, I agree)
   - Keluar (Exit)
3. Select "Ya, saya setuju".
4. The main menu appears.

---

## Main Menu

After agreeing, the bot shows 5 options via poll.

| No | Option (Indonesian) | Focus Area |
|----|---------------------|------------|
| 1 | Apa itu stunting? | Stunting education |
| 2 | Kondisi anak saya | Child condition |
| 3 | Perasaan saya hari ini | Parenting stress |
| 4 | Latihan & tips hari ini | Daily practice |
| 5 | Bantuan langsung | Direct support |

Select an option by tapping it in the poll.

---

## Menu 1: Stunting Education

Trigger words: "stunting" or "apa itu"

The bot shows a poll with 6 sub-options:

1. Pengertian stunting (Definition)
2. Penyebab stunting (Causes)
3. Tanda-tanda stunting (Signs)
4. Dampak stunting (Impact)
5. Cara pencegahan (Prevention)
6. Kembali ke menu utama (Return to main menu)

After each topic, the bot asks if you want to read another.

Step-by-step:

1. From the main menu, select "Apa itu stunting?"
2. The bot shows the stunting sub-menu.
3. Select "Pengertian stunting".
4. The bot explains the definition and adds a reinforcement message.
5. Select "Ya, tampilkan submenu lagi" to go back to the sub-menu.
6. Select another topic or choose "Kembali ke menu utama".

What each topic covers:

- Pengertian stunting — definition, what it affects beyond height
- Penyebab stunting — 5 causes including nutrition gaps and repeated infection
- Tanda-tanda stunting — 6 signs, includes a tip to use the KMS growth card at Posyandu
- Dampak stunting — short-term and long-term effects
- Cara pencegahan — 6 prevention steps from exclusive breastfeeding to stimulation

---

## Menu 2: Child Condition

Trigger words: "kondisi" or "anak saya"

The bot shows a poll with 4 conditions:

1. Berat badan sulit naik (Weight not increasing)
2. Anak terlihat lebih pendek (Child appears shorter)
3. Nafsu makan kurang (Low appetite)
4. Sering sakit (Frequent illness)

Each response includes:

- A validation statement
- A brief list of causes
- One small action to try today
- A reinforcement message

After reading, select:

- "Kondisi lainnya" — returns to the condition list
- "Kembali ke menu utama" — returns to the main menu

Step-by-step:

1. From the main menu, select "Kondisi anak saya".
2. The bot shows the condition poll.
3. Select "Nafsu makan kurang".
4. Read the response.  The bot gives one small step to try today.
5. Select "Kondisi lainnya" to check another condition.
6. Or select "Kembali ke menu utama".

---

## Menu 3: Feelings (Parenting Stress)

Trigger word: "perasaan"

The bot shows a poll with 4 options:

1. Khawatir anak stunting (Worried about stunting)
2. Capek dan stres (Tired and stressed)
3. Merasa kurang maksimal (Feeling inadequate)
4. Bingung harus mulai dari mana (Unsure where to start)

Each response follows a 4-part structure:

1. Emotion validation — acknowledges the feeling without judgement
2. Normalisation — confirms other mothers feel this way
3. Reframing — shifts the perspective
4. Coping strategy — one or two actions to do right now

After reading, the navigation poll shows:

- "Perasaan lainnya" — returns to the feelings list
- "Stress Check-in" — opens the Stress Check-in feature
- "Latihan & tips hari ini" — goes to the daily practice menu (appears for some options)
- "Kembali ke menu utama"

Step-by-step:

1. From the main menu, select "Perasaan saya hari ini".
2. The bot shows the feelings poll.
3. Select "Capek dan stres".
4. Read the validation, normalisation, reframing, and breathing exercise.
5. Select "Stress Check-in" to open the check-in feature.

---

## Menu 4: Daily Practice

Trigger words: "latihan" or "tips"

The bot shows a poll with 5 options:

1. Memberikan 1 makanan bergizi hari ini (Give 1 nutritious food today)
2. Mengajak anak makan bersama (Eat together as a family)
3. Memperhatikan kebiasaan makan anak (Observe the child's eating habits)
4. Cek latihan saya (Check my practice)
5. Kembali ke menu utama

Select one practice.  The bot gives a guide and a reinforcement message.

What each practice includes:

- Makanan bergizi — a list of food options (egg, fish, tempeh, tofu, vegetables, fruit)
- Makan bersama — steps to make family mealtimes work, including turning off TV
- Memperhatikan kebiasaan — a list of observations to note and report to a health worker

### Sub-feature: Practice Check

Use this after you attempt a practice.

1. From the practice menu, select "Cek latihan saya".
2. The bot asks how your practice went.
3. Select one:
   - "Ya, berhasil" (Yes, it worked)
   - "Masih mencoba" (Still trying)
   - "Belum sempat" (Not yet)

Responses by selection:

- Ya, berhasil — the bot confirms progress and sends a booster message
- Masih mencoba — the bot encourages you to keep going
- Belum sempat — the bot acknowledges and offers to try again via a new poll

After the check, select:

- "Self-Efficacy Check" — opens the progress check feature
- "Latihan lainnya" — returns to the practice list
- "Kembali ke menu utama"

Step-by-step:

1. From the main menu, select "Latihan & tips hari ini".
2. Select "Mengajak anak makan bersama".
3. Read the guide and reinforcement message.
4. After trying it, return to the practice menu.
5. Select "Cek latihan saya".
6. Select "Ya, berhasil".
7. The bot sends a positive message.
8. Select "Self-Efficacy Check" to measure your progress.

## Extra Feature 1: Stress Check-in

Access this from:

- Menu 3 (Feelings) navigation — select "Stress Check-in"

The bot asks: "Hari ini Bunda merasa:" (How are you feeling today?)

Poll options:

1. Tenang (Calm)
2. Sedikit lelah (Tired)
3. Sangat stres (Stressed)

Responses by selection:

- Tenang — a reinforcement message and affirmation
- Sedikit lelah — a short rest strategy (deep breathing, water, 5-minute break)
- Sangat stres — a 5-step coping sequence, a reminder about asking for help, and the contact number

After the check-in, the navigation poll shows:

- "Self-Efficacy Check"
- "Bantuan langsung" (appears for stressed response only)
- "Kembali ke menu utama"

Step-by-step:

1. From Menu 3, select any feeling.
2. From the navigation, select "Stress Check-in".
3. Select "Sangat stres".
4. Read the 5-step coping sequence.
5. Select "Bantuan langsung" for the contact number, or "Kembali ke menu utama".

---

## Extra Feature 2: Self-Efficacy Check

Access this from:

- Menu 4 (Daily Practice) navigation — select "Self-Efficacy Check"
- Stress Check-in navigation — select "Self-Efficacy Check"
- Type: "self-efficacy check"

The bot asks: "Dibanding sebelumnya, Bunda merasa sekarang:" (Compared to before, how do you feel now?)

Poll options:

1. Lebih percaya diri (More confident)
2. Sama saja (No change)
3. Masih bingung (Still confused)

Responses by selection:

- Lebih percaya diri — a reinforcement message and return to main menu
- Sama saja — encouragement to try a practice, navigation to Menu 4 or main menu
- Masih bingung — guidance to start with one small step, navigation to Menu 4 or stunting education

This feature works as a pre/post measure for self-efficacy in your research data.

Step-by-step:

1. Complete a practice from Menu 4.
2. From the navigation, select "Self-Efficacy Check".
3. Select "Lebih percaya diri".
4. The bot confirms your progress.
5. The main menu reappears.

## Extra Feature 3: Self-Efficacy Booster

Send "semangat" or "booster" at any point in the conversation.

The bot sends one message at random from this list:

- Bunda sudah melakukan yang terbaik hari ini
- Perubahan kecil tetap berarti
- Bunda mampu merawat anak dengan baik
- Setiap usaha kecil Bunda sangat berarti untuk anak
- Bunda tidak harus sempurna, cukup konsisten
- Bunda mampu melalui ini pelan-pelan. Tetap semangat!

This works at all stages except the welcome stage and initial agreement stage.

## Notes for Researchers

This bot covers three intervention layers:

1. Knowledge — Menu 1 builds understanding of stunting
2. Self-efficacy — Menus 2 and 4, plus Booster and Self-Efficacy Check, build confidence
3. Stress reduction — Menu 3 and Stress Check-in address parenting stress

The Self-Efficacy Check (Feature 2) produces three data points usable for pre/post measurement:

- More confident
- No change
- Still confused

Record these responses per user session to track shifts in self-efficacy over time.
